# Job Application Tracker

A simple web application built using Flask to track job applications and their current status.

## Features
- Add new job applications
- Update application status (Applied, Interview, Rejected, Offer)
- Delete applications
- View status summary dashboard

## Tech Stack
- Python
- Flask
- SQLite
- HTML

## How to Run
1. Install Python
2. Install Flask:
   pip install flask
3. Run the app:
   python app.py
4. Open browser:
   http://127.0.0.1:5000/

## Project Purpose
This project follows a real-world job application workflow where each job is added once and the status is updated as the hiring process progresses.
